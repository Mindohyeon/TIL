# JPA(Java Persistence API) 란?
- Java 에서 제공하는 API 이다. <b>스프링에서 제공하는 것이 아니다.</b>
- Java 어플리케이션에서 관계형 데이터베이스를 사용하는 방식을 정의한 인터페이스이다.
- ORM 이기 때문에 Java 클래스와 DB 테이블을 매핑한다. (sql 을 매핑하지 않는다.)
