# 변수

## var

- 읽기/쓰기가 모두 가능한 변수
- 할당 후 변경이 가능하다.

```kotlin
var a : Int = 10 // a = 10
var b : Char = 'a' // b = 'a'
```

## val

- 읽기만 가능한 변수
- 할당 후 변경이 불가능하다.

```kotlin
val b : Int = 10
b = 30  //오류 발생
```
